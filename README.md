# Searchbar / 搜索吧

Searchbar is a lightweight Edge extension that helps you switch search engines quickly and add your own custom search engines.

搜索吧是一个轻量的 Edge 扩展，帮你快速切换搜索引擎，并支持添加自定义搜索引擎。

## English

### Short description
Quickly switch search engines with support for custom engines.

### Store-style introduction
Searchbar makes it easier to jump between search engines while keeping the search flow simple.

- Switch between built-in search engines in one click
- Add custom search engines with your own URL template
- Reuse the current page keyword when switching to another engine
- Open a compact popup to search directly from the toolbar
- Manage visible engines, reorder them, and remove ones you do not need
- Works with a clean floating panel on search pages

Searchbar is designed for people who compare results across multiple search engines and want a faster workflow than copying keywords manually.

## 中文

### 简短介绍
轻松切换搜索引擎，支持自定义搜索引擎。

### 商店风格介绍
搜索吧让你在不同搜索引擎之间切换更顺手，尽量保留连续搜索的体验。

- 一键切换内置搜索引擎
- 支持通过链接模板添加自定义搜索引擎
- 切换时可复用当前页面关键词，减少重复输入
- 工具栏弹窗可直接输入关键词发起搜索
- 支持管理可见引擎、拖拽排序、删除不需要的引擎
- 在搜索页提供简洁的浮动面板，使用更直接

适合经常对比多个搜索引擎结果，或者希望把站内搜索、通用搜索统一到一个入口里的用户。

## Features

- Built-in and custom search engine support
- Toolbar popup for quick search
- In-page floating switcher
- Search engine ordering and visibility management
- Chinese and English localization
